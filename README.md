## Projets principaux

### Programmation Scientifique

L'essence de ces deux projets est d'utiliser un programme informatique au service de la résolution de problèmes de mécanique quantique :
- **[Oscillateur Harmonique Quantique 1D](https://github.com/Loic393/oscillateur-harmonique)** : On souhaite calculer les solutions de l'équation d'un oscillateur harmonique quantique à une dimension.
- **[Densité Local d'un Noyau](https://github.com/Loic393/nuclear-system-local-density)** : On souhaite calculer de manière optimisé la densité locale d'un système nucléaire et d'en montrer une représentation graphique. 

Technologies : C++, CMake, Doxygen, Armadillo

### Compilateur de Pseudo-Pascal

L'objectif du **[projet](https://github.com/Loic393/pseudo-pascal-compiler)** est d'implémenter les vérifications qu'un compilateur effectue lors de son utilisation. Le programme vérifie si un fichier est bien écrit en pseudo-Pascal, s'il n'y a pas d'erreur de _syntaxe_, de _type_ ou de _scope_.

Technologies : Ocaml, Ocamllex, Ocamlyacc

### Bibliothèque de Multithreading

Le but de ce **[projet](https://github.com/Loic393/my-thread)** est de comprendre en profondeur le fonctionnement de la gestion des _threads_ en implémentant sa propre bibliothèque C.

Technologie : C

### Application de rencontre

Technologies : Android Studio, Java, Maven, Google Authentification, Firebase Storage, Firestore Database
